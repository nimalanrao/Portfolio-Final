## ✨ Key Features

- **Liquid Glass UI**: A custom-built design system leveraging heavy backdrop blurs and subtle borders for a premium feel.
- **Interactive Experience Roadmap**: A scroll-triggered vertical timeline with alternating cards and bridge narrative text.
- **Advanced Physics Animations**: Every card and button features bouncy spring physics and 3D tilt effects powered by Framer Motion.
- **Custom Video Player**: A minimalist, high-performance video player modal with scrubbing, speed control, and silent playback.
- **Responsive Mastery**: Perfectly optimized for every screen size, from mobile devices to large desktop monitors.

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/nimalanrao/Portfolio-Final.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
